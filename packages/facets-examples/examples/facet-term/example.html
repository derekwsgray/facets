Definition

facet-term facets are standalone HTML components that render a linear gauge and optionally a label, value and annotation:


facet-term facets can be stacked to create a list:


The data schema for a facet-term is defined as:

export interface FacetTermData {
    ratio: number;
    label?: string;
    value?: number | string;
    annotation?: string;
    metadata?: any;
}
different ways of binding data to facets are explored in the working with js example

ratio is a number in the range [0,1] that describes the the value of the linear gauge in the facet.
metadata is provided as a convenience field where users can bind any value for later use either through JavaScript, a templating engine or the facet-template element. This concepts are further explored in the working with js and facet-template examples.

Customization

As described in the facet-blueprint example, facet-term instances support customization through the slots inherited from their parent facet-blueprint element.

facet-term expose a the following unique slots:
label - replaces the default label renderer.
annotation - replaces the default annotation renderer.
value - replaces the default value renderer.
bar - replaces the default bar renderer. It is usually not recommended to use this slot.
