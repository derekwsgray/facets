Definition

facet-histogram facets are standalone HTML components that render a collection of values as a histogram:


facet-histogram facets can be stacked with themselves or other facet types to create a list:


The data schema for a facet-histogram is defined as:

export interface FacetHistogramBucketData {
    ratio: number;
    values?: (number | string)[];
    metadata?: any;
}

export interface FacetHistogramData {
    buckets: FacetHistogramBucketData[];
    metadata?: any;
}
different ways of binding data to facets are explored in the working with js example

buckets is an array of descriptors (FacetHistogramBucketData) for each bucket to be rendered.
ratio is a number in the range [0,1] that describes the the value to be rendered for the current bucket.
metadata is provided as a convenience field where users can bind any value for later use either through JavaScript, a templating engine or the facet-template element. This concepts are further explored in the working with js and facet-template examples.
