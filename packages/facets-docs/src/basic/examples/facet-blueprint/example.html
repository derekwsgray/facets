facet-blueprint

The facet-blueprint element represents the building blocks of facets, it implements the basic layout for every facets and their content, each facet-blueprint includes the following 5 slots:

header
footer
left
right
content

header, left, right and footer are not used by facet implementations and are reserved to hold user customizations.
content is where facets will render their elements.

facet-blueprint elements can be, and often are, nested:

outer header
outer footer
outer left
outer right
outer content

inner header
inner footer
inner left
inner right
inner content

Since every facet is built on top of a facet-blueprint it is safe to assume that they expose the same 5 slots than a facet-blueprint.

Access to each slot is provided through the slot property defined in the shadow DOM API .

In JavaScript, developers can specify the default behaviour of each section by extending the FacetBlueprint class (or any other facet class) and overloading the methods:
renderContent
renderHeader
renderFooter
renderLeft
renderRight
