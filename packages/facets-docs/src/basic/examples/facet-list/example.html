Definition

facet-list is a standalone HTML component that renders a list of any facet instances, as described in its data attribute:


facet-list components behave as a div element, meaning that they use the full width of its parent element:

In the example above, the parent container of the facet-list element has a width of 80%, resize the window to see how the facet-list reacts.

The data schema for a facet-list is defined as:

export interface FacetListDataElement {
    type: string;
    data: any;
}

export type FacetListData = FacetListDataElement[];
different ways of binding data to facets are explored in the working with js example

type is the tag name of the facet that will be instantiated i.e. facet-term
data is the data that will be bound to the facet instance.
